# AAFACT Python Wrapper

This is a lightweight Python wrapper for the **compiled AAFACT application**.
It does NOT reimplement any algorithms in Python.

## Requirements
- Python 3.9+
- Compiled AAFACT executable (Windows/Linux/macOS)
- MATLAB Runtime (matching the version used to compile AAFACT)

## Installation (editable)
```bash
cd python_wrapper
pip install -e .
